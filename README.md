## ⚙️ Instalación del proyecto  

1. 🔽 Clonar el proyecto:  
   `git clone https://github.com/MartinFaraguna/PP2-Standalone.git`  

2. 📦 Instalar dependencias:  
   `npm install`  

3. ▶️ Ejecutar la app:  
   - Si tienes Ionic CLI:  
     `ionic serve`  
   - Si no tienes Ionic CLI:  
     `npm run start`  
     La aplicación correrá en: http://localhost:4200/
